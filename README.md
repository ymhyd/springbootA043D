### 五、项目介绍

项目编号：springbootA043D

设备维护管理系统是用来高效管理和优化设备维护工作，以减少故障、延长设备寿命并降低运营成本。

角色：管理员、用户

管理员：登录、系统首页、个人中心、用户、用户身份、部门、岗位名称、设备信息、设备报修、维修台账、设备巡检、任务信息、合同档案、经营收入、经营支出、系统公告。

用户：注册、登录、系统首页、个人中心、设备信息、设备报修、维修台账、设备巡检、任务信息、合同档案、经营收入、经营支出。
